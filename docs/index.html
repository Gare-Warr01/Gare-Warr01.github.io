Tracking 20 Years of U.S. Flu Outbreaks

U.S. Influenza Trends (2000–2023)

“Data doesn’t lie — the flu always comes back.”

Welcome to the U.S. Influenza Tracker, an interactive website analyzing over two decades of influenza surveillance data from the Centers for Disease Control and Prevention (CDC).

Project Objective

The purpose of this project is to provide a clear, interactive, and data-driven overview of influenza trends in the U.S. over the past 20 years. This website was created to demonstrate practical skills in:

  • Data wrangling and analysis in R
  • Interactive data visualization (Plotly, Leaflet)
  • Building and deploying a website using R Markdown and GitHub Pages

The goal is to make complex public health data accessible to a general audience.

Data Highlights

  • Time Period: 2000–2023
  • Total Specimens Analyzed: Over 10 million
  • Key Metrics: Influenza A & B cases, % positive, ILI percentages

Explore the Dashboard

  • View flu trends by year and virus type
  • See national average ILI (Influenza-like Illness) percentages
  • Explore an interactive U.S. map of flu cases
  • Browse raw data and download for your own analysis
  • Learn about the project creator

Data Source

All data used in this project was obtained from the CDC FluView Surveillance System: https://www.cdc.gov/flu/weekly/fluviewinteractive.htm
